تطبيق NuxFlix - تعليمات الرفع على GitHub وبناء APK تلقائي
=============================================================

اسم الباكج: com.nuxflix.app
اسم التطبيق: NuxFlix

---------------------------------------------------------
الطريقة 1: رفع المشروع عبر تطبيق على الهاتف (مثل "Working Copy"
أو "Termux + git" أو تطبيق GitHub الرسمي)
---------------------------------------------------------

1) أنشئ Repository جديد فارغ على GitHub (بدون README ولا .gitignore)
   مثلاً باسم: nuxflix

2) فك ضغط هذا الملف (NuxFlix.zip) في مجلد على الهاتف

3) ارفع كل محتوى المجلد NuxFlix (بما فيه مجلد .github المخفي!)
   إلى الـ Repository اللي أنشأتها

   تنبيه مهم: مجلد .github يبدأ بنقطة، وهو مجلد مخفي.
   بعض تطبيقات رفع الملفات ما تظهرش المجلدات المخفية بشكل افتراضي،
   تأكد أن إعدادات التطبيق تعرض "Hidden files" قبل الرفع، أو
   استعمل تطبيق يدعم git بشكل مباشر (زي Termux) للتأكد
   أن .github/workflows/build.yml تم رفعه فعلاً.

---------------------------------------------------------
الطريقة 2: عبر Termux (الأسهل والأضمن)
---------------------------------------------------------

1) ثبت Termux من F-Droid
2) نفذ الأوامر التالية بالترتيب:

   pkg install git -y
   cd storage/downloads        # أو المسار اللي فيه NuxFlix بعد فك الضغط
   cd NuxFlix
   git init
   git add .
   git commit -m "NuxFlix first commit"
   git branch -M main
   git remote add origin https://github.com/USERNAME/nuxflix.git
   git push -u origin main

   (بدّل USERNAME باسمك على GitHub، وسيطلب منك اسم المستخدم
   وTOKEN بدل الباسوورد - أنشئ Personal Access Token من
   GitHub Settings > Developer settings > Personal access tokens)

---------------------------------------------------------
بعد الرفع
---------------------------------------------------------

1) روح لتبويب "Actions" فوق فصفحة الـ Repository على GitHub
2) بش تلقى workflow اسمه "Build APK" يشتغل تلقائياً
   (يستغرق حوالي 2-4 دقائق)
3) بعد ما يخلص (علامة ✅ خضراء)، ادخل للـ workflow run
4) تحت فقسم "Artifacts" بش تلقى ملف اسمه "NuxFlix-debug-apk"
   حمّلo، فيه ملف APK جاهز للتركيب مباشرة على أي هاتف أندرويد
